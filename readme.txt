Форма авторизации uLogin через социальные сети. Улучшенный аналог loginza.

== Description ==

uLogin — это инструмент, который позволяет пользователям получить единый доступ к различным Интернет-сервисам без необходимости повторной регистрации,
а владельцам сайтов — получить дополнительный приток клиентов из социальных сетей и популярных порталов (Google, Яндекс, Mail.ru, ВКонтакте, Facebook и др.)

== Installation ==

1. Скопируйте папку uLogin в application/maxsite/plugins/
2. Активируйте модуль 
	Для этого зайдите в "панель администратора" -> "настройки" -> "плагины". Выбирите "uLogin" и нажмите "Включить".

== plugin login_form ==

Для удобства авторизации есть плагин login_form, в котором присутствует интеграция формы uLogin. 
	
Активации плагина 
	зайдите в "панель администратора" -> "настройки" -> "плагины". Выбирите "login_form" и нажмите "Включить".
